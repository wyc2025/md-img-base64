- 🛡️ 智能跳过代码块中的图片引用
- 🛡️ 智能跳过行内代码中的图片引用
- 🔍 防止误转换文档中的示例代码

### 改进

- 🧪 增强代码块检测算法
- 🧪 完善测试用例，验证代码块保护功能
- 📝 更新 README.md，说明代码块保护特性

### 技术细节

使用占位符替换法保护代码块：
- 先提取所有代码块（`````）替换为占位符
- 再提取所有行内代码（`code`）替换为占位符
- 处理完图片后恢复代码块内容

### 示例

```markdown
# 正常转换
![图片](./image.png)  ✅ 会被转换

# 代码块保护
\`\`\`javascript
// 这里的图片不会被转换
![代码中的图片](./image.png)  ⛔ 不会被转换
\`\`\`

# 行内代码保护
这里有一段 \`![行内代码中的图片](./image.png)\`  ⛔ 不会被转换
```

## [1.1.0] - 2026-03-02

### 新增功能

- ✨ 新增对 HTML `<img>` 标签的支持
- ✨ 支持多种引号格式：双引号、单引号、无引号
- ✨ 支持复杂的 img 标签属性组合
- ✨ 改进日志输出，区分 Markdown 图片和 HTML img 标签

### 支持的语法

现在支持以下所有格式：

#### Markdown 语法
```markdown
![alt text](./images/pic.png)
```

#### HTML img 标签（双引号）
```html
<img src="./images/pic.png" alt="图片">
```

#### HTML img 标签（单引号）
```html
<img src='./images/pic.png' alt='图片'>
```

#### HTML img 标签（无引号）
```html
<img src=./images/pic.png width=200>
```

#### HTML img 标签（复杂属性）
```html
<img width="200" height="100" src="./images/pic.png" alt="图片" class="image">
```
